# Dynamic Job Board

A responsive job board built with **React and Tailwind CSS** that allows users to browse and filter job listings dynamically.

## Features

* View job listings with details like title, department, location, and role type
* Filter jobs by department
* Real-time job search
* Fully responsive design

## Tech Stack

* React (Vite)
* Tailwind CSS

## Installation

1. Clone the repository

```
git clone https://github.com/tanujgupta18/job-board.git
```

2. Install dependencies

```
npm install
```

3. Start the development server

```
npm run dev
```
